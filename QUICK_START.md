# 🚀 Quick Start Guide

## ⚡ Get Started in 5 Minutes

### Step 1: Start Backend (Terminal 1)
```bash
cd backend
npm install
npm start
```
✅ Server should run on `http://localhost:5001`

### Step 2: Start Frontend (Terminal 2)
```bash
cd frontend
npm install
npm run dev
```
✅ App should open on `http://localhost:5173`

### Step 3: Create Account
1. Open browser to frontend URL
2. Click "Register"
3. Fill in: Name, Email, Password
4. Click "Register"

### Step 4: Login
1. Enter your email and password
2. Click "Login"
3. You'll be redirected to Dashboard

### Step 5: Explore Features
1. **Create Workspace** - Use sidebar form
2. **View Storage** - Click "Storage" button
3. **Create Tasks** - Go to Tasks tab
4. **Send Messages** - Go to Messages tab

## 🎯 Key Features to Try

### 3. Create Channel
- Go to "Channels" tab
- Enter channel name
- Select workspace
- Click "Create Channel"

### 4. Manage Tasks
- Go to "Tasks" tab
- Create new task
- Move between columns (To Do → In Progress → Done)

### 5. Send Messages
- Go to "Messages" tab
- Select channel
- Type message
- Click 🚀 Send
