# Team Workspace App - Frontend Setup

## Manual Directory Creation Required

Please create the following directory structure manually:

```
frontend/
  src/
    api/
    context/
    hooks/
    components/
    pages/
    routes/
    styles/
```

Run these commands in your terminal:

```bash
cd frontend
mkdir src
cd src
mkdir api context hooks components pages routes styles
cd ..
```

Or on Windows Command Prompt:
```cmd
cd frontend
mkdir src
cd src
mkdir api
mkdir context
mkdir hooks
mkdir components
mkdir pages
mkdir routes
mkdir styles
cd ..
```

After creating the directories, I will provide all the source code files.

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

The app will be available at http://localhost:5173
